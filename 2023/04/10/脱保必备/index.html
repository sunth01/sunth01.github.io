    Pomise

    • 三个状态 成功/失败/初始化
    • promise.all() // 所有的 promise 全部成功则返回成功，其中一个失败就返回失败
    • promise.race() // 对比返回最快的 promise，如果最快的 promise 失败，则失败，race 只管最快的，不管其他的 promise
    • promise.resolve() // 直接创建一个从成功的 promise
    • promise.reject() // 直接创建一个失败的 promise

    栈和堆的区别

    • 栈是一段连续的内存空间，用于存储局部变量、函数参数和返回地址等。而堆是另一种内存空间，专门用于动态分配内存和存储复杂数据结构。
    • 在栈中，系统自动分配和释放内存空间，不需要手动管理内存。在堆中，开发者需要手动管理内存，包括申请内存、释放内存和防止内存泄漏等。
    • 栈是一种简单的数据类型，只支持基本数据类型和引用类型的指针。堆则可以存储任意类型的数据，包括对象、字符串、数组等。
    • 由于栈是连续的内存空间，因此访问速度很快。而堆是分散的内存空间，访问速度较慢，特别是在处理大型数据结构时。
    • 栈中的局部变量和函数参数在函数调用结束后会立即被销毁，而堆中的数据则可以在任何时候被使用和修改，并且需要手动释放内存。
